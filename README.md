#  Calculatrice — JavaScript

##  Description
Cette calculatrice permet d’effectuer des opérations arithmétiques de base via une interface web simple.  
Le projet met l’accent sur la logique métier et la gestion de l’état en JavaScript.

##  Objectifs techniques
- Gestion des entrées utilisateur
- Implémentation de la logique mathématique
- Interaction HTML / JavaScript

##  Technologies utilisées
- HTML5
- CSS3
- JavaScript

##  Fonctionnalités
- Addition
- Soustraction
- Multiplication
- Division
- Réinitialisation de l’affichage

##  Lancer le projet
1. Télécharger le projet
2. Ouvrir `index.html` dans un navigateur web

##  Auteur
Projet personnel démontrant la maîtrise des bases du JavaScript.
